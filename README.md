# 🚜 Agrinho Valley

O **Agrinho Valley** é um jogo de estratégia, gerenciamento agrícola e tem fins educacionais devido suas 50 perguntas, desenvolvido utilizando a biblioteca **p5.js**. O objetivo do jogador é gerenciar uma fazenda tecnológica de forma equilibrada, maximizando os lucros com a venda de safras enquanto mantém altos índices de sustentabilidade e responde a auditorias fiscais reais sobre práticas ecológicas.

---

## 🚀 Funcionalidades

* **Quadro de Missões Dinâmico:** Sistema de progressão estruturado em 5 fases de objetivos crescentes (plantio, colheita, controle de caixa, metas financeiras e sobrevivência temporal).
* **Simulação de Drone Inteligente:** Sistema de telemetria onde o drone umidifica os talhões de terra em tempo real ao passar o cursor sobre o grid.
* **Sistema de Economia Volátil:** O preço de mercado das sacas oscila diariamente, exigindo que o jogador decida o melhor momento estratégico para vender seu estoque.
* **Mecânica de Auditoria (Quiz):** 50 perguntas balanceadas baseadas nas diretrizes ambientais reais do Paraná (ADAPAR/IDR), com punições severas para infrações ecológicas.
* **Eventos Climáticos:** Chance de ocorrência da temida *Geada Negra*, simulando os impactos reais do clima na agricultura do Sul do país.
* **Interface Responsiva:** Adaptação dinâmica de escala para diferentes resoluções e suporte a botões touch para jogabilidade em dispositivos móveis.

---

## 🎮 Como Jogar

1. **Plantio:** Clique em qualquer talhão de terra arada (com umidade acima de 50%) para gastar uma semente e iniciar o cultivo.
2. **Irrigação:** Mova o cursor do drone sobre a plantação para umidificar a terra manualmente ou utilize a função "Regar Tudo" se o solo estiver em níveis críticos.
3. **Colheita:** Quando a planta atingir o estado "PRONTO" (cor dourada), clique sobre ela para colher e enviar as sacas para o estoque.
4. **Venda:** Monitore a variação de preço diária. Se a sua fazenda tiver alta sustentabilidade, você receberá bônus ecológicos nas vendas!
5. **Passar o Dia:** Use a função de avançar o tempo para processar o crescimento, mas fique atento às taxas diárias de manutenção.
